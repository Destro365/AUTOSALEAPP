# AUTOSALEAPP

An automotive e-commerce platform developed as a college project. The application allows users to browse, search, and manage vehicle listings through an intuitive interface.

## Key Features
* **User Authentication:** Secure Sign-up and Log-in system (powered by Firebase).
* **Smart Search:** Search bar for specific models or filtering by vehicle type icons.
* **Vehicle Catalog:** A dedicated "Vehicles" tab listing all available inventory.
* **News & Updates:** Stay informed with the latest automotive news.
* **Contact System:** Direct communication channel with the company.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript 
* **Backend:** Node.js
* **Database & Auth:** Firebase
